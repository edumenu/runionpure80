This is a nutrition/fitness website which has CMS system built behind it. The CMS system is built with PHP and will be used to change content on the website dynamically.

admin_page/ This folder contains all the files and folders for the admin page

   Login/ This folder contains all the files and folders for the login page. This includes the main login page, the registration page.
   
   admin/ This folder contains the main admin dashboard page.
      main/ This folder contains the main dashboard file.
      assets/ This folder contains all the css files for all the admin pages.
      includes/ This folder contains all the other PHP files for the admin page.
   
   bower_components/ this folder contains the jQuery and toastr bower components
     
   vendor/ This folder contains folder and files for composer
   
css/ This folder contains all the css files for the main wbesite, this includes the main and the shop page.

fonts/ This folder contains the fonts for all pages on the website

img/ This folder contains all images on the website 

includes/ This folder contains the product list and modal for the shop page on the website.

js/ This folder contains javaScrpt files for the website 

scripts/ This folder contains a script file for updating the modal for each product selected

scss/ This folder contains all the scss files for the website.

index.php/ This file contains the code for the main website page

shop.php/ This file contains the code for the shop page
